# Сложно сосредоточиться

Адаптивный одностраничный сайт с поддержкой светлой и тёмной темы, автоматическим переключением по системным настройкам и ручным управлением через UI.

## Цели проекта
- Сверстать сайт по макету с учётом Mobile First
- Реализовать две цветовые схемы (dark / light)
- Обеспечить автоматическое и ручное переключение тем
- Отработать CSS-переменные, гриды и адаптивную типографику

## Используемые технологии

| Категория | Решение |
|---------|--------|
| Разметка | HTML5 (семантическая) |
| Стили | CSS3 |
| Сетки | CSS Grid |
| Адаптивность | Mobile First, media queries |
| Типографика | `clamp()` |
| Темы | CSS Variables, `color-scheme` |
| JS | переключение темы |
| Деплой | GitHub Pages |

## Ключевые реализации

### Темизация
- Основная тема — тёмная
- Цвета, отступы и шрифты вынесены в `variables.css`
- Вторая тема переопределяет значения переменных
- Поддержка системной темы через `<meta name="color-scheme">`
- Принудительное переключение тем через JS (без изменения HTML-разметки кнопок)

### Адаптивность
- Подход Mobile First
- Брейкпоинты: `768px`, `1024px`
- Резиновая вёрстка без горизонтального скролла
- Адаптивные размеры заголовков через `clamp()`

### Вёрстка
- Основная раскладка на CSS Grid
- Логические CSS-свойства (`margin-block`, `inline-size`)
- Фиксированный фоновый слой (`background-attachment: fixed`)
- Переиспользуемые декоративные элементы через псевдоэлементы

### Доступность и UX
- Декоративные элементы скрыты от скринридеров (`aria-hidden`)
- Корректные `alt` у изображений
- Ленивая загрузка изображений (`loading="lazy"`)
- Фокус-стили через `:focus-visible`

## Структура проекта

| Файл | Назначение |
|----|-----------|
| `index.html` | Основная разметка |
| `globals.css` | Сброс и базовые правила |

| `dark.css` | Переопределения тёмной темы |
| `light.css` | Переопределения светлой темы |
| `script.js` | Переключение тем |
